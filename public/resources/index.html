Data Science with R

R for Data Science This book starts with data visualisation and transformation of data that’s already been imported and tidied which was perfect for me because my previous experience in R mostly involved importing and tidying data in order to create some sort of ABT (Analytical Base Table) which I would export to a csv-file for further manipulation in other programs, or running a quite simple model on ready data. I recommend you to start with chapters 1-8, followed by chapter 27 covering R Markdown and chapter 18 covering piping before you start with the rest.

Text mining and natural language processing with R

Text Mining with R: A Tidy Approach by O’Reilly assumes the reader is at least slightly familiar with the R packages dplyr, ggplot2, and the %>% “pipe” operator.

Text Analysis with R - For Students of Literature - Second Edition (2020) by author Matthew Jockers is quite good as a complement to O’Reilly’s book mentioned above.
